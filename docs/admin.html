API 使用文檔

📖 方法 1: 更新單支股票（部分字段）

PATCH /api/stock/HK.01801 
{
  "currentPrice": 92.05,
  "changePercent": -0.22,
  "assessment": "P2 平穩持倉"
}

📖 方法 2: 替換單支股票（完整覆蓋）

PUT /api/stock/HK.01801

📖 方法 3: 更新宏觀摘要

PATCH /api/summary 
{
  "hsi": 25096.46,
  "hsiChange": 0.82,
  "upCount": 3,
  "downCount": 2
}

📖 方法 4: 批量更新

POST /api/stocks/batch

📖 方法 5: 查詢數據

GET /api/portfolio — 獲取全部數據
GET /api/stock/HK.01801 — 查詢單支股票

📖 方法 6: 重置數據

POST /api/reset
